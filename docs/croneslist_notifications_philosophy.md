# Croneslist Notifications Philosophy

## Core Principle: Respect the Player's Attention

Notifications are **OFF by default**. The game should never feel like a mobile gacha trying to addict you. Players opt-in to exactly what they want.

## Why Optional Notifications Work Better:

### 1. **Player Agency**
- YOU decide when you're broom shopping
- YOU set what's worth interrupting your tarot readings
- YOU control your compact's behavior

### 2. **Meaningful Alerts**
When your compact DOES glow, it means something:
- That Moonweave you've been hunting for weeks
- A broom under $200 when you finally saved enough
- Not every piece of duct-taped garbage

### 3. **Natural Gameplay Patterns**
Players naturally check Croneslist when:
- Morning routine (coffee + Croneslist)
- After successful readings (have cash now!)
- During downtime between clients
- When walking in the rain (motivation!)

### 4. **Avoiding Notification Fatigue**
Bad: *bzzt* "Broom posted!" (50th time today)
Good: *GLOW* "MOONWEAVE $750!!!" (once a week)

## Implementation Details:

### First Time User Experience:
```
"Want to know when specific brooms post?"
[Set up notifications] [I'll check manually]
```

### Default Settings (All OFF):
- [ ] Regular brooms
- [ ] Premium brooms  
- [ ] Other items
- Max price: $250
- Search terms: []

### Recommended Settings:
- [x] Premium brooms (these are RARE)
- [x] Regular under $220
- [ ] Other items
- Search: ["moonweave", "stormchaser"]

### Notification Levels:
1. **Standard buzz**: Regular broom matching criteria
2. **Urgent glow**: Premium broom posted
3. **Visual only**: Notification dot on Croneslist icon

## Player Psychology:

### Without Notifications:
- Check Croneslist as part of routine
- Discover brooms organically
- Feel accomplished when catching one
- No pressure or FOMO from constant alerts

### With Smart Notifications:
- Only interrupted for REAL opportunities
- Each alert is exciting, not annoying
- Can focus on tarot/life without worry
- Trust the system to alert for the good stuff

## The Sweet Spot:
Players checking Croneslist 3-5 times daily naturally, with notifications only for the specific broom they're saving for. When that compact glows for a Moonweave at $800, it's a MOMENT, not noise.
