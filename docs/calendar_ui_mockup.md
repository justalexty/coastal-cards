## Special Visual Effects

### Moon Phase Colors
- **New Moon**: Deep purple outline (🌑)
- **Full Moon**: Silver glow effect (🌕)
- **Regular phases**: Soft gray

### Holiday Colors
- **Lunar New Year**: Gold/red with particle effects
- **Equinoxes**: Season colors (pink/green/orange/blue)
- **Samhain**: Purple/orange spooky vibes
- **Beltane**: Pink/red romantic theme

### Interactive Elements
- Click calendar to see full month details
- Hover over moon phase for power description
- Click holidays for lore/bonus details
- Rent label links to financial summary

## Minimal Mode (During Readings)
Just shows: `Mar 1 🌒` in corner, expands on hover

## Night Mode
Darker colors, moon phases glow brighter, holidays use softer colors
